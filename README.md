# Fanzo: Smart Stadium Experience Prototype

Fanzo is a high-fidelity, interactive web prototype designed to revolutionize the physical event experience for attendees at large-scale sporting venues. Built with React and Vite, the platform focuses on solving common stadium pain points like crowd movement, waiting times, and real-time coordination through an immersive, "glassmorphic" user interface.

## ✨ Key Features

The prototype simulates a comprehensive suite of smart-stadium integrations:

- **Live Density Dashboard:** Real-time crowd heatmap indicating busy gates and wait times for concessions and restrooms.
- **In-Seat Ordering:** A complete cart flow allowing fans to order food and merchandise for express locker pickup.
- **AR Wayfinding (Mock):** A conceptual augmented reality interface guiding fans directly to their assigned seats or groups.
- **Group Sync:** A social hub for tracking friends' locations and orchestrating meetups within the venue.
- **Live Multi-Angle Replays:** A mocked real-time media player capable of switching between exclusive stadium camera angles.
- **Transit & Parking Hub:** Real-time integrations for local Metro departures, ride-share surge pricing, and live parking lot availability.
- **Gamification System:** Attendees earn "Fanzo Points" for mitigating queue times or arriving during assigned windows.
- **Secure Ticketing & Wallet:** Authentication wrapper leading to personalized digital passes and stored payment options (Apple Pay / Stripe mock).

## 🛠️ Technology Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Custom Vanilla CSS featuring a premium "glassmorphism" aesthetic, CSS Grid/Flexbox, and dynamic micro-animations.
- **Icons:** `lucide-react`
- **Routing:** `react-router-dom`

## 🚀 Getting Started

To run this prototype locally on your machine:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/parkheegulati/Fanzo.git
   cd Fanzo
   ```

2. Install the required dependencies:
   ```bash
   npm install
   ```

3. Start the local Vite development server:
   ```bash
   npm run dev
   ```

4. Open the link provided in your terminal (typically `http://localhost:5173`) in your browser to interact with the prototype.
